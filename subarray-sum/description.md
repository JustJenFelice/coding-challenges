# Two Sum

Given an array of integers, return indices of the subarray such that they add up to a specific target.

## Example

### Input:

nums = [1, 5, 3, 6, -1, 13, 5]
target = 18

### Output:

**startIndex (inclusive), endIndex (exclusive)**
[1,1]

## Run the program

In the folder run the command:

```
python subarray-sum.py
```
